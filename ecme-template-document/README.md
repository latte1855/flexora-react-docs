# Ecme React Template 說明文件索引

本資料夾整理 Ecme React Tailwind Admin Template 的各項操作指南，所有前端開發者在修改 `flexora-react-ui` 前，必須先熟悉以下文件：

| 檔案 | 內容摘要 |
| --- | --- |
| `Ecme React Tailwind Admin Template — 開發者操作手冊.md` | Ecme 範本的啟動方式、專案結構介紹、Theme/Layouts 切換與常見指令。 |
| `ecme-react-template-guide.md` | 站在專案實務角度，說明如何引用 Ecme 組件、路由、權限，以及與 Flexora 現有架構整合的注意事項。 |
| `layout.md` | 版面配置技巧：Menu/TopBar/Content 區域如何組合、常用容器與 spacing、響應式設計。 |
| `form.md` | 表單元件使用方式、驗證、欄位排列建議以及與後端 DTO 的對應。 |
| `table.md` | 資料表/清單組件設定、欄位模板與自訂 render、分頁等。 |
| `styling.md` | Tailwind CSS 規範、色票/字級設計與 Ecme 提供的常用 utility。 |
| `components.md` | 共用 component 快速索引（按鈕、卡片、Sidebar、Drawer…）與使用範例。 |

> 📌 開發 UI 之前，請務必先閱讀上述文件並對 Ecme Template 有基本掌握，再開始調整 `flexora-react-ui` 畫面。
