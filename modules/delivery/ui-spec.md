# Delivery Module – UI Spec

初步規畫：

1. **Delivery Hub**：與其他模組一致的 Filter Hub + List（可分「待打包」、「待出貨」、「已出貨」）。  
2. **Detail Drawer**：顯示 Delivery Note 資訊、包裝列表、追蹤碼、Workflow 按鈕。  
3. **Package/Serial Drawer**：維護每箱內容、序號、重量。  
4. **關聯操作**：從 Sales Order / Warehouse 列表跳入 Delivery。
