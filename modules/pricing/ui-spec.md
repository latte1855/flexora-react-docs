# Pricing Module – UI Spec

暫定畫面：

1. **PriceRule List**：顯示條件、適用範圍、優先順序，可拖拉調整。
2. **Rule Drawer**：編輯條件（客戶群、ItemGroup、PriceList、期間）、計算公式。
3. **PriceList Assignment**：將價目表套用到某客戶 / 通路。
4. **Pricing Trace Viewer**：顯示一次試算的各 Rule 命中狀況。
