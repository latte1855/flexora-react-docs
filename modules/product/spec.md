# Product Module – Functional Spec

> 狀態：草稿（2025-12-03）  
> 目的：集中定義 Product、Item SKU、Pricebook 之間的資料模型與行為。詳細敘述將在 Phase 5 展開時補齊。

## 範圍

- Product（品項主檔）：描述產品線、品牌、屬性。
- Item SKU：對應實際銷售/採購單位，含 UoM、稅別、預設價。
- Pricebook / ItemPrice：不同幣別、通路的定價資料。
- 關聯模組：報價、庫存、採購、銷售訂單。
