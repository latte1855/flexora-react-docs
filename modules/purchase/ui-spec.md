# Purchase Module – UI Spec

> 暫時參考 `docs/ui/` 下的 `purchase-*` 或 Inventory 相關 wireframe；目前尚無正式搬遷。

預計畫面：

1. **Purchase Workspace (Hub + List)**：左側篩選 Rfq/PO 狀態，右側 List/Pipeline 顯示採購單。
2. **PO Drawer / Full Editor**：採購單基本資料、明細、Delivery Schedule、Workflow 操作。
3. **Receipt / Return Drawer**：顯示實際收貨與退貨細節。
